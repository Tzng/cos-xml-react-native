# cos-xml-react-native

Tencent COS XML SDK for React Native

## Installation

```sh
npm install cos-xml-react-native
```

## Usage

```js
import CosXmlReactNative from "cos-xml-react-native";

// ...

const result = await CosXmlReactNative.multiply(3, 7);
```
